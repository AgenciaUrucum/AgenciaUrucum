**`Desenvolvedor WordPress Sênior (PHP | HTML | CSS | JS | Elementor | JetEngine)`**

Sou **Cristiano Pedrosa Ferreira**, de **Oriximiná/PA**, com **+15 anos de experiência** em WordPress e **PHP** desenvolvendo sistemas complexos.
